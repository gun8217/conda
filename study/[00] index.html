복합대입연산자
  • += (덧셈 후 대입) 변수에 값을 더하고 그 결과를 다시 변수에 대입합니다.
    예시: a += 5는 a = a + 5와 같습니다.
  • -= (뺄셈 후 대입) 변수에서 값을 빼고 그 결과를 다시 변수에 대입합니다. 예시: a -= 3은 a = a - 3과 같습니다.
  • *= (곱셈 후 대입) 변수에 값을 곱하고 그 결과를 다시 변수에 대입합니다. 예시: a *= 2는 a = a * 2와 같습니다.
  • /= (나눗셈 후 대입) 변수의 값을 나누고 그 결과를 다시 변수에 대입합니다. 예시: a /= 4는 a = a / 4와 같습니다.
  • %= (나머지 연산 후 대입) 변수의 값을 나머지 연산하고 그 결과를 다시 변수에 대입합니다. 예시: a %= 3은 a = a % 3과 같습니다.
  • **= (거듭제곱 후 대입) 변수의 값을 제곱하고 그 결과를 다시 변수에 대입합니다. 예시: a **= 2는 a = a ** 2와 같습니다.
  • //= (정수 나눗셈 후 대입) 변수의 값을 정수 나눗셈하고 그 결과를 다시 변수에 대입합니다. 예시: a //= 5는 a = a // 5와 같습니다.
